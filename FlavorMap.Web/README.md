# FlavorMap — Frontend

Interactive web application for exploring Hanoi's culinary culture by season.

## Tech Stack
- React 18 + TypeScript
- Vite
- Framer Motion
- React Router v6

## Prerequisites
- Node.js 18+
- FlavorMap.Api backend running on http://localhost:5065

## Setup

1. Install dependencies:
   npm install

2. Create `.env` file in this directory:
   VITE_API_BASE=http://localhost:5065

3. Start development server:
   npm run dev

4. Open browser at:
   http://localhost:5173

## Notes
- Backend must be running before starting frontend
- Images are served from the backend's wwwroot directory
